### Comandos:
Para limpar o texto:
```
python cleaner.py
```
Para gerar o vocabulário a partir do texto limpo:
```
python vocab_generator.py
```
Para fazer a correção a partir do vocabulário:
```
python corrector.py "frase para correção"
```
